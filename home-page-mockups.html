Home page — initial directions

Eight static compositions for a future artisti7607.com home. Ideas 1–3 use the compact navy header bar; 4–5 use the spacious utility-button header (Headers · alternates); 6–8 pair those chrome types with mega menus (panels always open for wireframe review — see Headers · mega explorations).

Each block is a self-contained wireframe: square corners, brand navy / blue / neutrals. Copy is placeholder. Ideas 4–5 use spacious CTA headers without mega panels. Ideas 6–8 show how a home first screen can read when News, Servizi, or Formazione keeps a wide mega open (same patterns as the header mega mocks, plus hero or digest below).

1 · Hero stack + three pillars + news band

Single strong hero (navy + blue accent bar), then three equal entry points (News / Formazione / Collecting), then a shallow “In evidenza” row for two stories—mirrors how the live site surfaces NEWS and secondary areas without a carousel.
