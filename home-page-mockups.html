Home page — initial directions

Five static compositions for a future artisti7607.com home. Ideas 1–3 use the compact navy header bar; ideas 4–5 use the spacious utility-button header (see Headers mockups · alternates). Body layouts vary below the header.

Each block is a self-contained wireframe: square corners, brand navy / blue / neutrals. Copy is placeholder. Ideas 4–5 pair a light, padded header (FAQ / Ricerca as outline actions, Accedi as primary CTA) with a calmer body—same information architecture as the contract header, different density.

1 · Hero stack + three pillars + news band

Single strong hero (navy + blue accent bar), then three equal entry points (News / Formazione / Collecting), then a shallow “In evidenza” row for two stories—mirrors how the live site surfaces NEWS and secondary areas without a carousel.
