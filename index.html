Brand & accessory colors from assets/ + static neutrals / semantic

Muted captions on this page: The smaller line under each token name uses neutral-500 at 12px (~4.15:1 on white). That is below WCAG AA for normal body-sized text—it’s a deliberate demo choice. For production UI, use neutral-700 and/or 14–16px for anything people need to read comfortably. Each A11y strip under a swatch only rates contrast for that color as a background (or role where noted).
